# tftp_recovery
router recovery app and tftp server


TFTP Recovery Tool is a Material You desktop application designed for easy and quick recovery of bricked routers (especially Cudy WR300 and others) via TFTP flashing.

✨ Key Features
TFTP Server on port 69 — listens for the router’s request and transfers the firmware file without extra steps.

Automatic network setup — the app assigns a static IP 192.168.1.88 to your Ethernet port (with admin privileges) and reverts to original settings after recovery.

Selectable network interface — all Ethernet adapters are supported, manually chosen in settings.

Router model database — built-in hints and exact parameters for dozens of devices (Cudy, TP-Link, Xiaomi, etc.). Easily extend via models.json.

Verbose real-time log — all TFTP events, errors, and transfer status are displayed live in the bottom panel.

Dark Material You theme — the interface picks accent colors from your Windows wallpaper, fully dark, with animations and adaptive layout.

Bilingual UI — switch between Russian and English with one button.

Automatic updates — checks GitHub for new versions and notifies you when available.

Portable — single executable, no installation required, works on Windows and Linux.

🧩 Who Is It For
For ordinary users who have given up and are ready to throw away a dead router — no more memorizing commands, manually setting IP addresses, or searching for a TFTP client. Just select the folder with recovery.bin, plug in the cable, and press "Start Server".




TFTP Recovery Tool — это десктопное приложение в стиле Material You, созданное для простого и быстрого восстановления «кирпичных» роутеров (особенно Cudy WR300 и других моделей) через TFTP-прошивку.

✨ Ключевые возможности
TFTP-сервер на порту 69 — ожидает запрос от роутера и передаёт файл прошивки без лишних действий.

Автонастройка сети — приложение само назначит вашему Ethernet-порту статический IP 192.168.1.88 (с правами администратора), а после завершения восстановления вернёт исходные настройки.

Выбор сетевого интерфейса — поддержка всех Ethernet-адаптеров, выбирается вручную в настройках.

База моделей роутеров — встроенные подсказки по подключению и точные параметры для десятков устройств (Cudy, TP-Link, Xiaomi и другие). Можно легко дополнить файл models.json.

Максимально подробный лог — все события TFTP, ошибки и статус передачи отображаются в реальном времени в нижней панели.

Тёмная тема Material You — интерфейс подбирает акцентный цвет под обои вашего Windows, полностью тёмный, с анимациями и адаптивным дизайном.

Двуязычный интерфейс — русский и английский язык переключаются одной кнопкой.

Автоматическое обновление — проверка новых версий на GitHub и уведомление при обнаружении.

Переносимость — один исполняемый файл без установки, работает на Windows и Linux.

🧩 Для кого
Для обычных пользователей, которые уже отчаялись и готовы выбросить «умерший» роутер — больше не нужно запоминать команды, настраивать IP вручную или искать TFTP-клиент. Просто выберите папку с recovery.bin, подключите кабель и нажмите кнопку «Запустить сервер».
